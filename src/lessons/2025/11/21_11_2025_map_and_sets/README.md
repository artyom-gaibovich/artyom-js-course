### Краткое резюме

1) Set — коллекция уникальных значений, появившаяся в ES6.
2) Позволяет легко избавляться от дубликатов.
3) Имеет методы: add, delete, has, clear.
4) Свойство size показывает количество элементов.
5) Удобно перебирается через for...of.
6) Легко преобразовать в массив через оператор spread (...).
7) Чаще всего Set применяют для уникализации массивов и хранения уникальных данных.


### Краткое резюме по мапам


1) Map — коллекция пар ключ–значение с ключами любого типа.
2) Основные методы: set, get, has, delete, clear.
3) set() возвращает Map (тот же самый) → можно использовать чейнинг.
4) Для непримитивных ключей Map сравнивает по ссылке, а не по значению.
5) У Map есть удобное свойство size.
6) Отлично подходит, если нужны сложные типы в качестве ключей или строгий порядок элементов.
