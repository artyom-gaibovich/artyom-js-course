### Задание мини-задания для самостоятельной практики Set

1) Дан массив:
["apple", "banana", "apple", "orange", "banana"]
— Создайте Set и выведите количество уникальных фруктов.

2) Создайте Set и добавьте туда следующие значения:
"JS", "HTML", "CSS", "JS"
— Удалите "HTML" и проверьте, есть ли "React".

3) Напишите функцию getUnique(arr), которая принимает массив и возвращает массив уникальных значений.

4) Создайте Set из строки "hello" и преобразуйте его обратно в массив символов.



### Задание мини-задания для самостоятельной практики по Map 

1) Создайте Map для хранения города → население.
Добавьте 3 города, получите одно значение, удалите один город.

2) Используйте чейнинг, чтобы заполнить Map значениями:
"JS" → "JavaScript", "TS" → "TypeScript", "HTML" → "HyperText".

3) Создайте массив key = [1,2].
Используйте его как ключ в Map. Получите значение обратно.

4) Попробуйте вызвать map.get([1,2]) при установке map.set([1,2], "value").
Объясните, почему результат undefined.

5) Проверьте наличие ключей с помощью has().
