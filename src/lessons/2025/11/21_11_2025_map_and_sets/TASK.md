### Задание мини-задания для самостоятельной практики

1) Дан массив:
["apple", "banana", "apple", "orange", "banana"]
— Создайте Set и выведите количество уникальных фруктов.

2) Создайте Set и добавьте туда следующие значения:
"JS", "HTML", "CSS", "JS"
— Удалите "HTML" и проверьте, есть ли "React".

3) Напишите функцию getUnique(arr), которая принимает массив и возвращает массив уникальных значений.

4) Создайте Set из строки "hello" и преобразуйте его обратно в массив символов.
