Double DQN. Dueling Network. Функционал ошибки.

В стандартном Q-learning используется максимизация целевого признака

$Q_{\text{target}} = r + \gamma \max_{a'} Q(s',a')$,

где $Q(s',a')$ берется из той же функции, что и текущая оценка $Q(s,a)$. Это приводит к переоценке $Q$-значений, т.к. максимизация может усиливать ошибки.

Для устранения переоценки Double Q-learning разделяет выбор действий и оценку его ценности, используя две различные сети:

  1. Главная сеть $Q^A_{\mathbf{w}}$ для выбора действия:

    $a^* = \argmax{Q^A_{\mathbf{w}}(s',a')}$.

  2. Целевая сеть $Q^B_{\mathbf{w}}$ для оценки выбранного действия:

    $\hat{Q} = r + \gamma Q^B_{\mathbf{w}}(s',a^*)$.

Dueling Network

Идея: разделить оценку $Q(s,a)$ на две компоненты:

  1. функция ценности состояния $V(s)$: оценка "качества" состояния независимо от выбранного действия.
  2. функция преимущества $A(s,a)$: насколько действие $a$ лучше среднего действия в состоянии $s$:

    $Q(s,a) = V(s) + A(s,a) - \frac{1}{|A|}\sum A(s,a')$.
