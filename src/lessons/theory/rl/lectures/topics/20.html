Вариационные методы. $EM$-алгоритм. Вариационный вывод

Вариационные методы используются для приближенного вычисления сложных вероятностных распределений, когда аналитическое решение вычислительно невозможно. В основе подхода лежит идея аппроксимации истинного распределения более простой моделью.

Ключевые шаги:

  1. Определить сложное апостериорное распределение $p(z \mid x)$, которое мы хотим аппроксимировать.
  2. Выбирается семейство более простых распределений $q(z)$, которое будет использоваться для аппроксимации.
  3. Минимизация $KL$-дивергенции между $q(z)$ и $p(z \mid x)$. Данная минимизация эквивалентна максимизации вариационной нижней границы $ELBO$.

$EM$-алгоритм используется для нахождения параметров вероятностных моделей в условиях скрытых переменных. Алгоритм итеративно максимизирует правдоподобие наблюдаемых данных, прибегая к разделению задачи на два этапа: ожидание $E$ и максимизацию $M$.

Ключевые шаги:

  1. $E$-шаг

    Вычисление ожидания полной логарифмической функции правдоподобия, исп. текущие оценки параметров:

    $Q(\theta \mid \theta^{(t)}) = \mathbb{E}[\ln{p, z \mid \theta}]$,

    где $\theta^{(t)}$ — текущие параметры, $z$ — скрытые переменные.

  2. $M$-шаг

    Обновление параметров $\theta$ для максимизации функции $Q(\theta \mid \theta^{(t)})$:

    $\theta^{t+1} = \argmax{Q(\theta \mid theta^{t})}$.

  3. Повторение шагов 1-2, пока алгоритм не сойдется.
