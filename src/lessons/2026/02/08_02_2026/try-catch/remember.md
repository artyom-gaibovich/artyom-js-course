
### Try / catch vs then / catch

| then / catch | try / catch  |
| ------------ | ------------ |
| цепочки      | линейный код |
| вложенность  | читаемость   |
| сложнее      | проще        |



### Главное, что нужно запомнить

 - ✔ await при ошибке = throw
 - ✔ try / catch ловит ВСЕ ошибки внутри блока
 - ✔ finally выполняется всегда
 - ✔ fetch требует ручной проверки response.ok
 - ✔ throw new Error() — хорошая практика
